## Comience en cuatro simples pasos

Aprende en cuatro simples pasos cómo cargar un documento, registrar firmantes y completar tu primer proceso de firma digital.

### Paso 1: Iniciar un nuevo proceso de firma

Para iniciar un nuevo proceso de firma, presione el botón:

<img
  src="/images/crear-proceso/btn-nuevo-proceso.jpg"
  alt="Btn Nuevo Proceso"
  title="Btn Nuevo Proceso"
  style={{ width:"38%" }}
/>

Se cargara la siguiente pantalla: **Preparación del documento**

![Preparacion Documento](/images/crear-proceso/preparacion-documento.jpg)

### Paso 2: Carga el documento

Ingresa un nombre para el documento:

<img
  src="/images/crear-proceso/nombre-documento.jpg"
  alt="Nombre Documento"
  style={{ width:"42%" }}
/>

Selecciona el archivo que vas a enviar a firmar:

<img
  src="/images/crear-proceso/selecciona-documento.jpg"
  alt="Selecciona Documento"
  title="Selecciona Documento"
  style={{ width:"53%" }}
/>

Presiona el botón: **Siguiente**

### Paso 3: Agrega un firmante

<AccordionGroup>
  <Accordion title="Agregar datos del firmante" icon="user">
    Install the Mintlify GitHub app from your [dashboard](https://dashboard.mintlify.com/settings/organization/github-app).

    Our GitHub app automatically deploys your changes to your docs site, so you don't need to manage deployments yourself.
  </Accordion>
</AccordionGroup>

### Paso 3: Anexa campo de firma y envia

<Accordion title="Publish your docs" icon="rocket">
  1. Commit and push your changes.
  2. Your docs will update and be live in moments!
</Accordion>
